 for install playwright :
  npm init playwright@latest


 To set permission for npm :
  Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

find task : netstat -ano | findstr :9323
to kill : taskkill /PID 30576 /F

To find trace :
npx playwright show-trace test-results\third-login-test-Google-Chrome-retry1\trace.zip

To Run : 
npx playwright test ./tests/selecters.spec.js --headed --browser=chromium --channel=msedge or 
npx playwright test ./tests/selecters.spec.js --headed --project="Microsoft Edge"
